# TTS 文字转语音 实现方案

> 日期：2026-07-22  
> 状态：待实现  
> 关联：F-25 ASR 之后、`ai/` 目录重构之前；[F-25 ASR 设计](./2026-07-22-F25-音频转文字-ASR-设计.md)；[AI 目录重构里程碑](./2026-07-22-AI目录结构重构-里程碑.md)

## 1. 已确认的决策

### 1.1 UX 方案（用户选定）
- **选定：方案 C** — Phase 1 划词朗读 + Phase 2 独立「文字转语音」页；两套共用同一套合成能力。
- **增补（强制）：** 划词朗读 / TTS 入口是否开放由管理员按用户配置。管理员**固定可用**；非管理员**默认关闭**；管理员可在用户管理中手动打开。
- **用户主路径（有权限时）：**
  1. 文档/图片预览 → 划词 →「划词问答」→ 选音色 →「朗读」→ 面板内播放 / 下载 mp3  
  2. 侧栏「文字转语音」→ 粘贴文本 → 选音色 → 生成 → 试听 / 下载  
- **未选方案（归档）：** 仅 A（只有划词）；仅 B（只有独立页）。

### 1.2 技术方案（用户选定）
- **选定：路线 Y1** — `User.ttsEnabled` 用户级开关 + 服务端鉴权；`role === admin` 始终可用。
- **关键技术点：** Prisma 字段；Admin 用户列表开关；登录/`me` 下发 `can_use_tts`；硅基 `POST /v1/audio/speech` 同步返回 mp3；字数上限；Key 回退链对齐 ASR；MVP **不落库、不走 BullMQ**。
- **未选路线（归档）：** Y2 仅全局总开关（无法按人开放）；Y3 用户开关 + 全局紧急总闸（MVP 过重）。

## 2. 范围

### 2.1 In Scope
- 后端 TTS Provider + HTTP（音色列表、合成 mp3）
- `User.ttsEnabled` + Admin 开关 API + 列表展示
- `can_use_tts` 透出到前端（登录 / 拉取当前用户）
- 前端：`DocumentAiPanel` 划词朗读（仅划词模式、有选中、且 `can_use_tts`）
- 前端：独立页 `/tts`（侧栏入口，仅 `can_use_tts` 可见；无权限路由守卫回首页）
- 服务端对 `/ai/tts/*` 强制鉴权（藏 UI 不够）
- Provider 单测（mock fetch）+ 权限相关单测或手测清单

### 2.2 Out of Scope
- 存进网盘、长文异步分段合并、自定义克隆音色、双人播客
- VIP 默认开通 TTS（本期不绑 VIP）
- 全局紧急总闸（Y3）
- `ai/` 目录大搬家（等 ASR+TTS 都完成后再做）
- 文生视频

## 3. 用户操作对照

| 步骤 | 用户动作 | 界面反馈 | 失败/边界 |
|------|----------|----------|-----------|
| 1 | 管理员打开「用户管理」 | 表中可见「文字转语音」开关；管理员行固定开且不可关 | — |
| 2 | 管理员打开某普通用户开关 | 开关变为开；该用户刷新后可见入口 | 不能关管理员的「固定开」 |
| 3 | 无权限用户打开 PDF 预览划词 | **不显示**朗读控件 | 直调 API → 403 |
| 4 | 有权限用户划词 → 点朗读 | Loading → 内嵌 `<audio>` 可播；可下载 mp3 | 无选中提示；超字数提示；上游失败提示 |
| 5 | 有权限用户进侧栏「文字转语音」 | 文本框 + 音色 + 生成 → 试听/下载 | 超字数、空文本、403 |
| 6 | 无权限用户访问 `/tts` | 路由拦回首页 | API 仍 403 |

## 4. 技术设计

### 4.1 架构与数据流

```
[Admin 用户表开关] → User.ttsEnabled
                         ↓
[登录 / GET me] → can_use_tts = (role==admin) || ttsEnabled
                         ↓
[前端] 显示朗读 / 侧栏 TTS 页
                         ↓
[POST /api/ai/tts/speech] → assertCanUseTts → tts.provider.synthesizeSpeech
                         ↓
硅基 POST {baseURL}/audio/speech → audio/mpeg Buffer → 浏览器 Blob 播放
```

权限判定（后端统一）：

```ts
function canUseTts(user: { role: string; ttsEnabled: boolean }): boolean {
  return user.role === 'admin' || user.ttsEnabled === true;
}
```

### 4.2 API

| 方法 | 路径 | 请求 | 响应 | 鉴权 |
|------|------|------|------|------|
| GET | `/api/ai/tts/voices` | — | `{ success, data: { maxChars, voices: [{ id, label, gender }] } }` | JWT + `canUseTts` |
| POST | `/api/ai/tts/speech` | `{ text, voiceId?, speed? }` | **原始** `audio/mpeg` 二进制 | JWT + `canUseTts` |
| PATCH | `/api/admin/users/:id/tts` | `{ enabled: boolean }` | `{ success, data: { id, tts_enabled, can_use_tts } }` | JWT + `@Roles('admin')`；禁止关闭管理员 |

说明：
- `GET /api/admin/users`（`listUsers`）增加 `tts_enabled`
- 登录 `toUserDto` 与当前用户接口增加 `can_use_tts`（派生字段；普通用户不能自改）

### 4.3 数据 / 存储
- **新字段：** `User.ttsEnabled Boolean @default(false) @map("tts_enabled")` + migration
- **复用：** 不写 Storage、不写 BullMQ
- **环境变量：**
  ```env
  AI_TTS_BASE_URL=https://api.siliconflow.cn/v1
  # AI_TTS_API_KEY=  # 空则回退 VISION → EMBEDDING → AI_API_KEY
  AI_TTS_MODEL=FunAudioLLM/CosyVoice2-0.5B
  AI_TTS_MAX_CHARS=2000
  ```

### 4.4 前端
- `DocumentAiPanel/index.vue`：划词模式 + `canUseTts` → 音色 +「朗读」+ `<audio>` + 下载
- `views/tts/index.vue`（新建）：独立工具页
- `Sidebar.vue`：有权限显示「文字转语音」
- `admin/index.vue`：用户表 TTS 开关列
- `api/ai.ts`：`getTtsVoices` / `synthesizeSpeech`（Blob）
- `api/admin.ts`：`setUserTtsEnabled`
- `stores/auth.ts`：用户对象含 `can_use_tts`；路由守卫拦 `/tts`
- i18n：`preview.aiTtsSpeak` 等（zh-CN / zh-TW / en-US）

### 4.5 后端
- **新建：** `src/files/ai/tts/tts.provider.ts`、`tts.provider.spec.ts`、`files-ai-tts.service.ts`、可选 `tts-access.util.ts`
- **Controller：** MVP 挂现有 `files-ai.controller.ts` 的 `/ai/tts/*`，或独立 controller 并在 `files.module.ts` 注册
- **修改：** Prisma；`admin.service.ts` / `admin.controller.ts`；`auth.service.ts`；`files.module.ts`

### 4.6 错误与边界
- 无权限：403「未开通文字转语音」
- 空文本 / 超字数：400
- 非法 `voiceId`：回退默认或 400（实现时固定一种并单测）
- 上游失败：简短 message，风格对齐 ASR
- 对管理员 PATCH `enabled: false` → 400「管理员固定开放」

## 6. 验收标准

- [ ] 普通用户默认无朗读入口与侧栏 TTS；speech API → 403
- [ ] 管理员无需开关即可用；管理端对其开关禁用/固定开
- [ ] 管理员打开某用户后，该用户刷新可见且可合成
- [ ] 关闭后入口隐藏，API 403
- [ ] 划词可朗读、可下载；无选中/超字数有提示
- [ ] 独立页可试听/下载；无权限无法进入
- [ ] 划词翻译 / 文档索引 / ASR 无回归
